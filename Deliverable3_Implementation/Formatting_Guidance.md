### **Chapter Headings (Level 1)**

- **Style**: Bold
- **Examples**:
  - "1. Introduction"
  - "5. Software Architecture"
  - "6. Detailed Design"

### **Section Headings (Level 2)**
- **Font Size**: **13 pt**
- **Style**: Bold
- **Examples**:
  - "5.1 Overview"
  - "5.2 Subsystem Decomposition"
  - "6.3 Object Interaction"

### **Subsection Headings (Level 3)**
- **Font Size**: **11pt** (or 12pt if using Times)
- **Style**: Bold or Italic
- **Examples**:
  - "5.2.1 Presentation Layer"
  - "6.3.1 UC-001: Browse Restaurant Menus"

## ğŸ“Š Table Formatting

1. **Table Borders**: Use clear borders for readability
2. **Table Headers**: Bold and centered
3. **Table Text**: Same font size as body text (11pt Arial or 12pt Times)
4. **Table Captions**: Place above or below table, include "Table X:" prefix

**Example Table Format:**

