# MenuMap

A collaborative class project to create a comprehensive menu mapping application.

## Project Overview

MenuMap is a web application designed to help users discover, explore, and navigate restaurant menus efficiently. The application will provide an intuitive interface for browsing menu items, filtering by dietary preferences, and discovering new dining options.

## Project Milestones

1. **Team Setup** - Repository setup, role assignments, initial planning
2. **Project Plan** - Requirements gathering, architecture design, timeline creation
3. **Use Cases** - User story development, use case documentation
4. **UML Diagrams** - System architecture, class diagrams, sequence diagrams
5. **SRD Draft** - Software Requirements Document creation and review
6. **Final Submission** - Complete documentation, final presentation, code delivery

## Deliverables

- Software Requirements Document (SRD)
- UML Diagrams (Class, Sequence, Use Case)
- Project Timeline and Gantt Charts
- Meeting Minutes and Documentation
- Source Code Repository
- Final Presentation
